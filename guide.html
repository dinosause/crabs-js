Crabs.js is an open-source particle library with different types of particles, each with their own animations. This is a quick guide to using Crabs.js in your websites.

Setting Up

Before adding particles, add these lines of code to the head of your webpage:

<link href="crabs.css" rel="stylesheet">  <script src="crabs.js"></script>
